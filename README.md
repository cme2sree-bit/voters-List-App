# Voters List App

This project allows you to search the voter list for a specific ward using name, family name, or voter number. The data source is the Kerala State Election Commission's official list.

## Features

- Search by name or family name
- View voter number, ward, and related details
- (Future) See possible party wise vote status

## Data Source

- Book1.xlsx: Ward-wise voter list uploaded in this repository

## How to Use

1. Download or clone this repository
2. Find the file Book1.xlsx inside the repo (data folder or root)
3. Use the Python search script or any Excel functions to find voters by name or number

## Project Structure

- Book1.xlsx - Main voter list data
- backend/   - Search and API code (add here if you create scripts)
- frontend/  - (optional) UI code
